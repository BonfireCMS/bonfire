# Bonfire API

> RESTful API and site controller.

## Getting started

Install dependencies:

```bash
scripts/init
```

Start the postgres and redis services:

```bash
scripts/start
```

## Testing

Run all the tests:

```bash
npm test
```

Run the tests and watch for changes:

```bash
npm run watch
```
